# BackendExam
In this project i have done database creation and api creation and i have implemented get , post ,put and delete methods
steps to start with express and node
1. create a folder named backendexam
2. cd backendexam
3. now initilize your project with node modules  (command : npm init -y)
4. now add express for server and api routes (command : npm install express)
5. then install sqlite database for storing and retriving local data (command : npm install sqlite3)
6. now install jsonwebtoken for creation and verification of jwt and bcrypt for password storing (command : npm install jsonwebtoken bcrypt)
7. create server.js
8. here you go
